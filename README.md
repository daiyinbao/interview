# Java 后端场景面试记录

这个仓库用于整理和沉淀常见的 **Java 后端场景面试题** 与对应的 **实现思路/示例代码**。重点放在系统设计与工程化能力的表达，不绑定具体业务背景，便于复用与迁移。

## 目标

- 把面试中高频的“场景题”拆解成可讲清楚的技术模块
- 通过最小可行实现体现设计思路、边界条件与工程取舍
- 积累可复用的代码片段，快速回忆关键点

## 内容范围

- 大数据量处理（流式去重、限内存统计、批量处理）
- 并发与性能（锁、分段、无锁思路、吞吐/延迟权衡）
- 存储与持久化（文件格式、内存映射、checkpoint）
- 系统设计（容量估算、读写路径、故障与恢复）
- 可靠性（幂等、边界检查、异常场景）
- 可观测性（指标、日志、可定位性）

## 目录结构

- `01_delayed_task/`：延时任务相关场景
- `02_bitset/`：位图/位集相关场景

后续会按主题继续扩展：
- 排队/削峰
- 缓存一致性
- 账户/计数/签到
- 文件/日志处理

## 使用方式

每个子目录通常包含：
- 题目背景与思路说明（README）
- 关键实现代码（可编译运行）

建议结合以下角度阅读：
- 输入规模与约束
- 关键数据结构选择
- 复杂度与资源占用
- 出错时的恢复策略

## 说明

本仓库仅用于学习与面试准备，不涉及任何真实业务数据。

如果你有想补充的场景题，可以直接提需求，我会按“思路 + 最小实现”的方式补充。
